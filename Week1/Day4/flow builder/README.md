# Flow Builder and Automation 

## 1. What is Flow Builder?

Flow Builder is a Salesforce automation tool used to automate business processes without writing code. It helps create workflows, update records, send emails, display screens, and perform actions automatically.

---

# 2. Types of Flows

## Screen Flow

A Screen Flow is an interactive flow that collects input from users through screens.

### Features

* Displays forms and questions
* Takes user input
* Used for guided processes

### Example

Student registration form in a College Management System.

---

## Record Triggered Flow

A Record Triggered Flow runs automatically when a record is created, updated, or deleted.

### Features

* Fully automated
* No user interaction needed
* Executes in the background

### Example

Automatically updating available course seats after student enrollment.

# 5. Manual vs Automated Process

## Process: Student Fee Reminder

### Manual Process

* Staff manually checks due dates
* Messages are sent individually
* Requires continuous monitoring

### Problems in Manual Process

* Time consuming
* Human errors may occur
* Students may miss reminders
* Difficult to manage large data

### Automated Process Using Salesforce

* Automatic reminders are sent before due dates
* Faster and more accurate
* Reduces manual effort
* Improves efficiency and communication

---

# 6. Reflection – Why Automation Matters in Enterprise Systems

Automation helps enterprises improve productivity, reduce manual work, and increase accuracy. It saves time by handling repetitive tasks automatically and helps organizations manage large amounts of data efficiently. Automation also improves consistency, reduces errors, and provides faster business operations.
