# 1. Search Solution Basics

## What I Learned
I learned the difference between SOQL and SOSL and when to use each query language.  
I also learned how Salesforce search works using indexing, selective queries, search optimization, and custom search solutions.

## One Key Learning
SOSL is useful when we do not know which object or field contains the required data because it can search across multiple objects efficiently.

## One Doubt / Question
How does Salesforce internally rank search results when multiple records match the same keyword?

# 2. Agentforce 360 Platform Events Basics

## What I Learned
I learned about event-driven architecture and how Salesforce Platform Events help systems communicate asynchronously.  
I also learned how to create platform events and subscribe to them using Apex triggers.

## One Key Learning
Platform Events help decouple event producers and consumers, making applications scalable and easier to integrate.

## One Doubt / Question
What is the maximum retention period for platform event messages in Salesforce?
